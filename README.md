# DeepThought BA Assignment

## What's in this repo

This repository contains the research deliverables for the DeepThought Business Analyst assignment.

| File / Folder | Description |
|---|---|
| `companies.csv` | **Part A primary deliverable** – 25 shortlisted companies matching all criteria |
| `methodology.md` | How companies were found, sources used, and key learnings |
| `prompts.md` | Every AI/search prompt used during research, with guardrails and reasoning |
| `proposal-1000-companies.md` | **Part B Question 2** – Scaled approach write-up for 1000-company research |
| `sources/` | Raw data extracts from each primary source |
| `research-log/` | Full log of all 75-100 companies investigated (pass/fail + reasons) |
| `scratch/` | Working notes used during research (not part of final review) |

## How to read this repo

1. Start with **`methodology.md`** to understand the research approach and criteria used.
2. Review **`companies.csv`** for the final 25 shortlisted companies.
3. Check **`research-log/companies-investigated.csv`** for the full funnel of companies evaluated.
4. See **`research-log/disqualifications.md`** for detailed reasons why companies were excluded.
5. Browse **`sources/`** to inspect the raw data pulled from each directory/database.
6. Read **`prompts.md`** for transparency on AI-assisted research steps.
7. Read **`proposal-1000-companies.md`** for the Part B scaled-research proposal.
